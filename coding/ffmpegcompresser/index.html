FFmpeg Compressor Script

Console Script front end for FFmpeg compressing and basic remuxing for quick and easy video conversion. Perfect for making transportable video files for applications like discord, you can also set a max file size to automaticly cut the video when it gets to large. Aswell as trimming video by selecting starting time and ending times. The FFmpeg base support hardware accelerated AV1 and HEVC encoding on NVIDIA and Intel GPUs and you can select and update FFmpeg manualy to support more as the FFmpeg isn't Embedded. FFmpeg must be downloaded manually from FFmpeg.org

Setup

Download Compressor.lua and Compressor.bat from github. Also download executable files for ffmpeg and either put them in a PATH folder or place it in a folder with the Compresser.lua and .bat files. Then create shortcuts of Compresser.bat to each folder you'd like to run the commpresser from. To run the script drag and drop and video file onto the shortcutted .bat file and follow the promts. To enable advanced mode simply edit the compresser.lua and change the variables
